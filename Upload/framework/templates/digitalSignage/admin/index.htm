Digitale Schaukästen

Software Download
Einstellungsdatei Download

Systemvoraussetzungen:
  • Min. Windows 7 (nur 64 bit)
  • Deaktivierter Bildschirmschoner und Energieoptionen auf immer an (auch Display)
  • Internetzugang ohne Proxy
Funktionen:
  • Über den digitalen Schaukasten kann der Vertretungsplan angezeigt werden.
  • Es können Powertpoints angezeigt werden, die über das Portal hochgeladen werden können.
  • Es ist keine Energieverwaltung vorhanden. (z.B. Monitor ein / aus usw.)
Installation:
  • Einfach einen Autostart auf DigitalSignageLauncher.exe setzen
  • Die Einstellungsdatei (settings.properties) hier downloaden und in das Programmverzeichnis kopieren.
  • Für eine Unterstützung von Powerpoint:
    • Die Einstellungsdatei settings.properties in einem Editor (nicht Word) öffnen. (z.B. Notepad++)
    • Die letzte Zeile beinhaltet folgenden Text: pathToPowerpoint=
    • Ergänzen Sie die Zeile um den Pfad zur Datei POWERPNT.EXE. Verwenden Sie bitte doppelte Backslash Zeichen. Beispiel: pathToPowerpoint=C:\\Program Files (x86)\\Microsoft Office\\Office16\\powerpnt.exe
    • Speichern Sie die Datei ab.
Die Schaukastensoftware / Funktion ist derzeit im Beta Test
