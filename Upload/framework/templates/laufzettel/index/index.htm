Laufzettel für Schulaufgaben und Unterrichtsgänge

Unter diesem Menüpunkt werden Laufzettel verwaltet, die für Unterrichtsgänge und Schulaufgaben verwendet werden.
Der Vorgang ist einfach:
  • Sie erstellen einen Laufzettel und geben dabei an, wann der Unterrichtsgang oder die Schulaufgabe stattfindet.
  • Sie wählen alle betroffenen Klassen aus
  • Sie wählen alle betroffenen Stunden aus
  • Das System ermittelt automatisch aus dem Stundenplan die betroffenen Lehrer und informiert diese.
  • Das System fordert von allen betroffenen Lehrern eine Zustimmung
  • Für einen Unterrichtsgang können Sie direkt das Antragsformular für die Schulleitung ausgefüllt herunterladen
  • Für eine Schulaufgabe können Sie den Übersichtzettel herunterladen
